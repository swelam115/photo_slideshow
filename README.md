# Photo Slideshow

Simple Picture Slideshow Program written in C.

## Features
- Loads images from a specified directory
- Shuffles and displays images with fade transitions
- Supports navigation with next, back, and pause functionality

## Requirements
- SDL2
- SDL2_image
